# سورس ميوزك سينو - Music Bot Source Sino

![Music Bot](https://source.unsplash.com/featured/?music,bot)

بوت تليجرام متطور لتشغيل الموسيقى في المكالمات والقنوات، مع لوحة تحكم، أزرار شفافة، وأوامر متعددة اللغات.

## الميزات
- تشغيل الموسيقى في المكالمات والقنوات.
- تحميل من يوتيوب، سبوتيفاي، إلخ.
- أزرار شفافة للتحكم.
- رسالة ترحيب مع صور.
- لوحة مطور: إذاعة، اشتراك إجباري.
- خروج تلقائي للحساب المساعد.

## التثبيت
1. استنسخ المشروع: `git clone https://github.com/yourusername/music-bot-sino`
2. قم بتثبيت المتطلبات: `pip install -r requirements.txt`
3. أضف المتغيرات في `.env` أو البيئة.
4. شغل: `python main.py`
